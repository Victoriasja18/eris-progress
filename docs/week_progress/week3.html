Minimap2

Let’s run the mappy without any eris and see what is the result.

After reading the minimap2 API, it seems minimap2 required tuning for specific data type (i.e., differ between long vs short reads, ONT vs pac-bio)

import mappy as mp
import pandas as pd

# Obtain the query first 
# !bedtools getfasta -fi data/EC958_complete_genome.fasta \
#     -bed data/complete_ec958_IS.reanotated.sorted.bed -s -name > data/IS_elements.fa

# Path
comp_fa_path = "Eris/data/EC958_complete_genome.fasta"

# Align the complete genome assembly fasta file
comp_fa_aln = mp.Aligner(comp_fa_path, preset = "map-ont")

#Read IS sequences and map them
with open("../result/minimap2_IS.bed", "w") as bed:
    for is_name, is_seq, _ in mp.fastx_read("../../data/IS_elements.fa"):
        for hit in comp_fa_aln.map(is_seq):
            print(hit)
            if not hit.is_primary:
                continue
            strand = "+" if hit.strand == 1 else "-"
            bed.write(f"{hit.ctg}\t{hit.r_st}\t{hit.r_en}\t{is_name}\t{hit.mapq}\t{strand}\n")
print("It's done!")
