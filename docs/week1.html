Convert the Fasta into GFA format

Since the complete genome assembly is in .fasta format, we need to convert it into .gfa format to run Eris. The .gfa format has an important information where we can see the overlaps, paths, and coverage, while the FASTA lack of this information.

We used minigraph to convert the fasta file into gfa format. The command is as follows:

minigraph -xggs data/EC958_complete_genome.fasta > data/EC958_complete_genome.gfa

# rerun Eris with the new gfa file
uv run eris -i ../data/EC958_complete_genome.gfa -d ../data/IS.mmi -o ../output/complete_gfa/eris_complete_gfa

Weakness of this might be not real topological information of the genome assembly, as the gfa file is generated from the fasta file.

Fragmentation of the complete genome assembly from the fasta file

I want to try create a fragmentations of the complete genome assembly to see if it can improve the results. As the complete genome assembly has 3 sequences, I will try to create 150 fragments for each sequence.

# Create fragments of the complete genome assembly
from Bio import SeqIO
from Bio.SeqRecord import SeqRecord
import random

complete_fasta = "../data/EC958_complete_genome.fasta"
fragmented_fasta = "../data/EC958_complete_genome_fragmented.fasta"

def fragment_genome(input_fasta: str, output_fasta: str, num_fragments: int = 150):
    records = list(SeqIO.parse(input_fasta, "fasta"))
    fragments = []
    
    for record in records:
        seq_length = len(record.seq)
        fragment_size = seq_length // num_fragments
        
        for i in range(num_fragments):
            start = i * fragment_size
            end = (i + 1) * fragment_size if i < num_fragments - 1 else seq_length
            fragment_seq = record.seq[start:end]
            fragment_id = f"{record.id}_fragment_{i+1}"
            fragments.append(SeqRecord(fragment_seq, id=fragment_id, description=""))
    
    SeqIO.write(fragments, output_fasta, "fasta")
    print(f"Written {len(fragments)} fragments to {output_fasta}")


# Fragment the complete genome assembly
fragment_genome(complete_fasta, 
    fragmented_fasta, 
    num_fragments=150)
Written 450 fragments to ../data/EC958_complete_genome_fragmented.fasta
